# Text-Based Game

This repository contains an early Python project from my Computer Science program. The application is a command-line adventure game where the player moves through rooms, collects items, and tries to reach the final objective.

## Project Overview

The game uses a room-based map and prompts the player to move between locations. As the player explores, they collect required items before entering the final room. The project demonstrates basic programming logic, user input handling, conditional statements, loops, and simple game-state management.

## Skills Demonstrated

- Python fundamentals
- Command-line user interaction
- Dictionaries and structured data
- Conditional logic and loops
- Input validation
- Basic game-state tracking
- Step-by-step problem decomposition

## How It Works

The player enters movement commands such as north, south, east, or west. The program checks the current room, validates the requested move, updates the player location, and tracks whether the required items have been collected.

This project represents foundational programming work and shows early practice with control flow, structured data, and interactive application logic.
